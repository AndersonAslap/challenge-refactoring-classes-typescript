> Sobre o GoRestaurant

<p>
O GoRestaurant é uma plataforma de pedidos de pratos
Lá pode cadastrar os pratos, remover e alterar e verificar
qual está disponível ou indisponível.
</p>

<br>

<p>
<ul>
  <li>React</li>

  <li>Html</li>
  <li>SASS</li>
</ul>
</p>
